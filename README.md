### **Environment Set**
1. Linux System
2. Make sure you have C Language installed
3. Clone the repo
***

### **How to play this Game**
#### Clone this repo
```sh
git clone https://github.com/SunGj921028/CSIE-Final_Project.git
```
#### Run the make command in the folder /catan_universe
```sh
make
```
#### Run the executable file
```sh
.catan
```
